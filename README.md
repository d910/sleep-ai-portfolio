# 睡眠AI应用开发作品集 🌙

围绕「睡眠健康」场景，使用Coze、dify、Python、RAG、Fastapi、Streamlit等技术开发的4个AI应用项目。

## 项目列表

### 📱 [01. 助眠智能体Agent](./01_sleep_chatbot/)
**技术**：Coze + Prompt Engineering + 插件 + 工作流 + 知识检索  
**功能**：温柔专业的睡眠辅导，实现与用户的简单交流，将复杂的助眠问题交给工作流处理  
**亮点**：多节点工作流分析用户当前情绪，通过网络搜索或检索知识库为用户生成个性化助眠建议

### 📚 [02. AI睡眠健康顾问](./02_AI睡眠健康顾问-Dify/)
**技术**：Dify、Chatflow、Qdrant、RAG
**功能**：生成基于睡眠医学文档的睡眠改善方案  
**亮点**：多轮对话 + RAG + 搜索降级
**体验链接**：[点击查看演示视频](./02_AI睡眠健康顾问-Dify/演示视频.mp4)

### 🌐 [03. 睡前内容生成器（Web应用）](./03_sleep_content_app/)
**技术**：vue + fastapi + 通义千问API  
**功能**：睡前故事生成 + 冥想引导（带语音）+ 睡眠日记模板  
**在线体验**：[点击访问](http://43.99.30.193/)  
**亮点**：带UI的完整Web应用，部署在云端，支持语音播放

### 🌐 [04. 睡眠互助社区（Web应用）](./04_sleep_community/)
**技术**：langchain + Chroma + RAG检索 + Streamlit + 通义千问API
**功能**：基于睡眠医学文档和用户经验的智能检索问答 + 助眠经验分享  
**在线体验**：[点击访问](https://sleep-community-5cfr8bnehvbxdvxjhyyrn9.streamlit.app/)  
**亮点**：基于langchain1.0+架构，带UI的完整Web应用，部署在云端

## 技术栈

- **无代码平台**：Coze，dify
- **编程语言**：Python 3.12
- **AI API**：通义千问（文本生成 + 语音合成）
- **核心技术**：Prompt Engineering / RAG / Agent / TTS / fastapi

### 开发工具
- Python、Streamlit
- Git、GitHub
- Docker（Dify本地部署）

## 项目背景

这是我在学习AI应用开发过程中的系列实战项目。选择「睡眠健康」作为主题，是因为：
1. 有真实的用户需求和应用场景
2. 能展示AI在垂直领域的应用能力
3. 从简单到复杂，逐步深入AI技术
