# 睡前内容生成器（Web应用）

## 项目简介
使用Python、Streamlit、通义千问API制作的助眠web应用Demo

## 功能特点
- 睡前故事生成
- 冥想引导（带语音）
- 睡眠日记模板

## 技术栈
- **平台**：streamlit
- **核心技术**：
  - Python 
  - Streamlit
  - 通义千问API
  - 语音合成
