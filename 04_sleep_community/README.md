# 睡眠互助社区（Web应用）

## 项目简介
使用langchain + Chroma + RAG检索 + Streamlit + 通义千问API制作的 知识库检索助眠问答web应用Demo

## 功能特点
- 基于睡眠医学文档和用户经验的智能检索问答
- 助眠经验分享

## 技术栈
- **平台**：streamlit
- **核心技术**：
  - langchain
  - Python
  - RAG 
  - Streamlit
  - 通义千问API
