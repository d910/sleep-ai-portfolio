# 睡眠AI应用开发作品集 🌙

围绕「睡眠健康」场景，使用Coze、Python、Streamlit等工具开发的3个AI应用项目。

## 项目列表

### 📱 [01. 助眠智能体Agent](./01_sleep_chatbot/)
**技术**：Coze + Prompt Engineering + 插件 + 工作流 + 知识检索  
**功能**：温柔专业的睡眠辅导，实现与用户的简单交流，将复杂的助眠问题交给工作流处理  
**亮点**：多节点工作流分析用户当前情绪，通过网络搜索或检索知识库为用户生成个性化助眠建议

### 📚 [02. 睡眠知识库](./02_sleep_knowledge_base/)
**技术**：Coze + RAG  
**功能**：基于睡眠医学文档的知识问答系统  
**亮点**：精准检索文档内容，提供专业睡眠建议

### 🌐 [03. 睡前内容生成器（Web应用）](./03_sleep_content_app/)
**技术**：Python + Streamlit + 通义千问API + 语音合成  
**功能**：睡前故事生成 + 冥想引导（带语音）+ 睡眠日记模板  
**在线体验**：[点击访问](https://sleep-ai-app-fgd6iyx2dxyamnsdkbihia.streamlit.app/)  
**亮点**：带UI的完整Web应用，部署在云端，支持语音播放

## 技术栈

- **无代码平台**：Coze
- **编程语言**：Python 3.12
- **AI API**：通义千问（文本生成 + 语音合成）
- **Web框架**：Streamlit
- **核心技术**：Prompt Engineering / RAG / Agent / TTS

## 项目背景

这是我在学习AI应用开发过程中的系列实战项目。选择「睡眠健康」作为主题，是因为：
1. 有真实的用户需求和应用场景
2. 能展示AI在垂直领域的应用能力
3. 从简单到复杂，逐步深入AI技术
